# Polygon Tool GIS

Детский инструмент для генерации карт провинций на основе географических данных.

## Описание

**Polygon Tool GIS** — это десктопное приложение для автоматической генерации карты провинций. Оно использует алгоритм Poisson Disc Sampling для равномерного распределения точек по карте и диаграмму Вороного для создания границ провинций.

Приложение работает с географическими данными из OSM-файлов и автоматически учитывает природные объекты: реки, озёра, заливы и береговые линии.

## Возможности

### 1. Импорт GIS-данных

Поддерживаемые форматы файлов:
- GeoJSON (.geojson, .json)
- OpenStreetMap (.osm)
- GeoPackage (.gpkg)

Извлекаемые данные:
- **Населённые пункты** — villages, cities, towns
- **Реки** — водные артерии (waterway = river) (пока не работает)
- **Береговые линии** — coastline (пока не работает)
- **Заливы** — bay
- **Озёра** — water bodies

### 2. Преобразование координат

Приложение автоматически преобразует географические координаты (широта/долгота в EPSG:4326) в пиксели для отображения на карте, используя проецию EPSG:3857 (метры).

### 3. Генерация провинций

Алгоритм работает следующим образом:

1. **Poisson Disc Sampling** — генерирует равномерно распределённые точки по всей карте с учётом минимального расстояния между ними
2. **Фильтрация по водным объектам** — точки не размещаются внутри озёр и заливов
3. **Проверка на суше** — использует глобальную маску суши/воды для корректного размещения точек
4. **Диаграмма Вороного** — строит границы провинций на основе полученных точек

### 4. Визуализация

Результат отображается с помощью matplotlib с подсветкой:
- Красные точки — исходные населённые пункты
- Синие точки — сгенерированные фоновые точки
- Границы провинций — линии диаграммы Вороного

## Установка

### Требования

- Python 3.13+
- Poetry (менеджер зависимостей)

### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Grayholm/polygon_tool_GIS.git
cd polygon_tool_GIS
```

2. Установите зависимости через Poetry:
```bash
poetry install
```

3. Активируйте виртуальное окружение:
```bash
poetry shell
```

4. Запустите приложение:
```bash
python main.py
```

## Использование

### Шаг 1: Импорт файла

1. Запустите приложение
2. Перейдите на вкладку **GIS file**
3. Введите размер карты (пикселей на метр)
   - Рекомендуемое значение: 0.01
   - Диапазон: 0.001 — 5.0
4. Нажмите **Import GIS file** и выберите файл с географическими данными

### Шаг 2: Генерация провинций

1. Перейдите на вкладку **Province Image**
2. Введите минимальное расстояние между точками
   - Меньшее значение = больше провинций
   - По умолчанию: 90
   - Диапазон: 10 — 500
3. Нажмите **Generate Province Map**
4. Дождитесь завершения генерации

## Зависимости

- **PyQt6** — графический интерфейс
- **GeoPandas** — работа с географическими данными
- **Shapely** — геометрические операции
- **NumPy** — числовые вычисления
- **SciPy** — научные вычисления (Voronoi, KDTree)
- **Matplotlib** — визуализация
- **global-land-mask** — определение суши/воды по координатам

## Пример использования

В папке `example_input/` находится пример файла `Sicilia_example.geojson` с данными для острова Сардиния (Италия). Вы можете использовать его для тестирования приложения.

## Текущие ограничения

- Границы провинций не обрезаются по береговой линии и линиям рек(в разработке, хотя возможно и не будет добавлено, слишком громоздко получится)
- Поддерживаются только населённые пункты типа village, city, town
- Озёра фильтруются по площади (> 5 км²)
